# 煮酒三国·化学兴汉

## 时代背景

东汉末年，建安年间（约公元200年）。天下大乱，群雄割据。曹操挟天子以令诸侯，雄踞北方；孙氏据江东；刘备颠沛流离，寄人篱下。百姓流离失所，白骨露于野，千里无鸡鸣。此乃中国历史上最混乱、最血腥，也最英雄辈出的时代之一。

## 核心设定

一个现代化学家意外穿越至建安五年（公元200年）。他没有任何外挂系统，只有脑中的化学与物理知识。在这个一切都要靠铁与血说话的时代，知识就是最锋利的武器。他需要从零开始：制取黑火药、改进炼钢、提纯青霉素、制造肥皂、提纯食盐——用化学改变战争的形态，也改变千万百姓的命运。

## 科技体系

第一层级：基础化学——提纯硝石、硫磺，制造黑火药；改进炼铁术，制备钢铁；提纯食盐，制皂。第二层级：中级应用——火绳枪、简易手榴弹、改进弩机、基本药物（如青霉素粗提物）。第三层级：高级应用——早期大炮、合金钢、更精密的化工流程。每一步都受限于当时的材料、工具和人力，不能一蹴而就。

## 社会风貌

士族门阀垄断知识与仕途，寒门难有出头之日。天下人以孝廉举荐为正途，视方术道术为异端。一个满口'化学方程式'的怪人，在这个时代要么被视为疯子，要么被视为仙人——而两者都极其危险。主角必须在'奇技淫巧'与'经世致用'之间找到平衡，在士族的蔑视与军阀的觊觎中生存。

## 地理格局

以刘备集团的视角展开：从新野小县起步，经赤壁之战夺荆州，取益州，北伐中原。战争场景涉及水战、攻城战、野战，每种形态都需要不同的化学武器和战术配合。关键地点：新野、襄阳、赤壁、江陵、成都、汉中、长安、洛阳。
