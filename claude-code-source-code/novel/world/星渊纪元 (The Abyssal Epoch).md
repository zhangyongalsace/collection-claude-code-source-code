# 星渊纪元 (The Abyssal Epoch)

## 宇宙概览

公元3047年，人类文明已扩展至银河系三分之一的区域，建立了横跨数千光年的星际联邦「泛星同盟」（Panstellar Concord）。然而，在银河系核心方向，一片被称为「星渊」的异常区域正在缓慢扩张——那里物理法则被扭曲，时间流速不均匀，空间本身在坍缩。星渊的边界被称为「事件穹界」，任何越过穹界的探测器都传回了令人不安的数据：星渊内部存在结构化的信号，暗示着某种古老而庞大的智慧。

人类目前处于「第三次星际扩张」的尾声，掌握了曲率引擎、意识上传、基因重塑等技术，但尚未突破光速壁垒。银河系中已发现至少12个已灭绝的外星文明遗迹，但从未与活着的智慧物种接触过——这被称为「大沉默悖论」。

## 文明格局

泛星同盟（Panstellar Concord）：人类主流政权，总部设在地球轨道上的「天枢环」。采用议会制，由7个核心星区和数百个边疆星区组成。名义上统一，实则各星区高度自治，利益博弈激烈。

深渊学院（The Abyssal Academy）：独立于同盟的科研组织，专注于星渊研究。总部在距星渊最近的「守望者空间站」。拥有银河系最先进的物理实验室和最庞大的数据库。被同盟视为既是盟友也是潜在威胁。

铸魂者集团（Soulforgers Consortium）：垄断意识上传和数字永生技术的巨型企业。表面上提供服务，暗中收集上传者的意识数据，用于构建一个被称为「蜂巢」的集体意识实验。

边缘自由联盟（Frontier Free Alliance）：边疆星区的松散联合，反对泛星同盟的中央集权。资源丰富但技术落后，是走私、黑市和灰色地带的温床。

静默教团（Order of Silence）：一个神秘的宗教组织，崇拜星渊中未知的智慧体。认为「大沉默」并非因为没有外星文明，而是因为所有文明最终都选择了「沉默」——融入星渊。

## 核心科技

曲率引擎（Warp Drive）：通过压缩局部空间实现超光速航行，但无法穿越星渊的扭曲空间。最高速度约为光速的500倍。

意识晶格（Consciousness Lattice）：将人类意识完整映射为量子态数据，可实现意识上传、备份和转移。但每一次上传都会产生微小的「意识漂移」，多次上传后的人格是否还是原来的「自己」成为深刻的哲学争议。

基因重塑（Gene Reshaping）：可以重写人类基因组，实现极端的环境适应。在殖民不同星球的过程中，产生了大量「亚种」人类——适应高压的深星人、适应零重力的虚空族、适应强辐射的焰裔等。

创物质合成（Exotic Matter Synthesis）：人造负质量物质，用于稳定虫洞和构建星门。极度稀缺，是银河系最有价值的资源。

星门网络（Stargate Network）：由创物质构建的固定虫洞，连接主要星区。是银河系贸易和军事的生命线，但星门的位置决定了各星区的战略价值。

## 星渊之谜

星渊并非自然现象。在银河系中心附近发现的「创世碑」——一块横跨数百公里的古老结构体——记录了一个被称为「编织者」（The Weavers）的超维文明的存在。编织者曾经统治整个银河系，在至少20亿年前建造了星门网络的基础设施。

星渊是编织者留下的「作品」——一个正在缓慢展开的高维结构在三维空间的投影。随着它扩张，三维空间的物理法则被逐步改写。星渊内部的信号是编织者的「回声」，一种跨越数十亿年的时间胶囊。

更令人不安的是，星渊的扩张速度在加快。按照当前趋势，它将在约300年内吞没最近的人类殖民地，3000年内覆盖整个银河系。人类必须在有限的时间内做出选择：逃离银河系、理解并控制星渊，或找到与编织者沟通的方式。

## 社会与文化

身份危机时代：意识上传技术使「死亡」变得可选，但也引发了深刻的身份认同危机。法律定义「人格连续性」为判断一个人是否仍然「存在」的标准，但这个定义本身就充满争议。

亚种分化：基因重塑使人类物种正在分裂。不同星球环境的人类之间的基因差异已经大到自然繁殖困难。一些激进派主张彻底分道扬镳，成为不同的物种；保守派则呼吁维护人类统一性。

信息鸿沟：核心星区与边疆星区之间的技术差距巨大。核心星区的居民享受数字永生和完美健康，边疆星区的居民还在为基本的医疗和辐射防护而挣扎。这种不平等是银河系最主要的矛盾来源。

大沉默的恐惧：所有已灭绝的外星文明都在相似的科技水平上消失了——略高于人类当前水平。这被称为「大过滤器的阴影」，是悬在人类头顶的达摩克利斯之剑。

## 关键地点

天枢环（Celestial Pivot）：地球轨道上的巨型空间站，泛星同盟的政治中心。容纳3000万人口，是人类历史上最大的人造结构。

守望者空间站（Watcher's Keep）：距星渊最近的人类前哨，深渊学院总部。持续监测星渊扩张，收容所有从星渊边缘回收的异常数据。

创世碑（The Genesis Monument）：银河系中心附近的古老结构体，编织者文明唯一已知的实体遗迹。表面刻满了尚未完全破译的符文，已破译的部分揭示了星渊的本质和编织者的存在。

幽荧星（Umbral）：星渊边缘一颗被事件穹界半吞噬的行星。其上有一个深渊学院的秘密研究基地，正在尝试直接从星渊内部提取数据。所有在那里工作的人员都报告了相同的经历——在星渊的方向「听到了什么」。

熔炉星（The Crucible）：铸魂者集团的工业中心，一个被完全改造的超级地球。整个星球就是一台巨大的量子计算机，用于运行「蜂巢」集体意识。
