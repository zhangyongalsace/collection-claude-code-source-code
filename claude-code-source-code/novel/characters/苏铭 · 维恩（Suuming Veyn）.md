# 角色：苏铭 · 维恩（Suuming Veyn）

## 身份

深渊学院首席研究员，星渊语言学奠基人

## 年龄

147岁（通过意识晶格续命，身体约40岁状态）

## 外貌

身材瘦削，面容清冷，左眼是一枚仿生义眼——在幽荧星事故中失去原眼。义眼呈暗金色，能在量子频谱范围内视物。常穿深灰色的学院制服大衣。

## 性格

极其聪明但不善交际，对学术有近乎偏执的执着。表面冷静自持，内心深处被星渊中的「声音」困扰——她是唯一声称能听懂星渊信号的人，但这也让她被部分人视为疯子。

## 背景

出身核心星区的学术世家，28岁加入深渊学院，35岁在幽荧星研究中遭遇事故失去左眼，却因此获得了感知星渊信号的异常能力。此后她致力于破译星渊信号，提出了震惊学界的「编织者语言假说」。她的发现引起了铸魂者集团的注意和静默教团的狂热追随。

## 核心冲突

她听到的星渊「声音」究竟是真正的外星通讯，还是意识晶格续命带来的幻觉？她必须在被政治博弈吞噬之前找到答案。

## 关系

与铸魂者集团的高层有隐秘的过去；被静默教团视为「先知」；与泛星同盟军方代表关系紧张。
