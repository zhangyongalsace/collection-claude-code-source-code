# 角色：奥伦 · 石川（Orun Ishikawa）

## 身份

边缘自由联盟的传奇飞行员和地下运输者，绰号「星尘」

## 年龄

34岁（焰裔亚种人类）

## 外貌

深红色皮肤，瞳孔如同液态金属，头发是细密的纤维状结构可以散热。这是焰裔适应强辐射星球的典型特征。驾驶时穿着破旧的飞行夹克，上面挂满了各个星区的通行徽章。

## 性格

洒脱不羁、幽默风趣，但在轻松的外表下藏着深深的愤怒。对核心星区的虚伪深恶痛绝，但也不认同边缘自由联盟的激进独立主张。是一个理想的行动派——在所有人争论的时候，他已经在行动了。

## 背景

出生在靠近星渊的一个边疆殖民地，父母是矿业工人。殖民地因星渊扩张被泛星同盟放弃，数万人被抛弃在即将被吞没的星球上。奥伦是少数逃出的人之一，从此对同盟彻底失望。他成为星门网络中最出色的走私者，以能穿越任何封锁而闻名。

## 核心冲突

他的故乡正在被星渊吞噬，而那里还有他不愿离开的族人。他必须在个人情感和更大使命之间做出选择。

## 关系

与韩洛有私交（韩洛曾试图救援奥伦的殖民地但被上级阻止）；被静默教团视为潜在的目标——他们想利用奥伦穿越星渊边界。
