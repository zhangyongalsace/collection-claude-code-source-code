# 角色：韩洛 · 卡斯特（Hanlo Kast）

## 身份

泛星同盟第七舰队指挥官，星门防御体系的实际负责人

## 年龄

58岁（自然衰老，拒绝意识上传）

## 外貌

高大魁梧，面容刚毅，满头白发但精神矍铄。右臂是军用义肢——在一次边疆冲突中被边缘自由联盟的游击队炸断。

## 性格

务实、强硬、不信任知识分子和企业。信奉实力和平秩序。对泛星同盟的官僚体制深恶痛绝，但仍然是它最忠诚的捍卫者。

## 背景

出生于边疆星区，少年时代经历了边疆叛乱，亲眼目睹了核心星区对边疆的漠视。从军后一路升迁，成为同盟军方最强硬的鹰派。主张对星渊采取军事封锁策略，反对深渊学院的「接触」研究。

## 核心冲突

他最害怕的不是外星人，而是人类自己——他确信铸魂者集团的蜂巢实验和边缘自由联盟的独立运动比星渊更可能摧毁人类文明。

## 关系

与苏铭对立（军方vs学术界）；暗中与边缘自由联盟的某些领袖有私人联系；不信任铸魂者集团。
