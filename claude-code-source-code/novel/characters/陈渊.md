# 角色：陈渊

## 身份

穿越者，原为某大学化学系副教授，专攻物理化学与无机化学。现化身'陈渊'，一个自称游学天下的布衣书生。

## 年龄

穿越时三十五岁，正值壮年。

## 性格

沉默寡言，内心极为理性冷静，但在目睹乱世惨状后，逐渐被一种沉重的使命感压住。他有现代人的良知与学养，却也因此与这个格格不入的时代。他不是英雄，只是被命运扔进炼炉里的一块铁——被烧着、被锤打着，慢慢变硬。

## 外貌

中等身材，面容清瘦，眉目间有一种与这个时代格格不入的疏离感。穿着粗布衣衫时像个落魄的书生，但那双眼睛——带着实验室里养出来的精确与冷淡——出卖了他。

## 能力

精通化学各分支：无机化学、有机化学、物理化学、分析化学。掌握基本的冶金学、药学知识。动手能力极强——他做过太多实验了。劣势：不通兵法，不擅权谋，不谙骑射。他所有的本事都在脑子里和手上。

## 穿越原因

一次实验室爆炸事故。他以为自己是死了，醒来时躺在一片焦土上，身上只有烧焦的实验服和脑中挥之不去的化学方程式。

## 口头禅或思维习惯

遇到问题时，脑子里会自动浮现化学方程式。害怕时，会默念实验步骤来让自己冷静。
