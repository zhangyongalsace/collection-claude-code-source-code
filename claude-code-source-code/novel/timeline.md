# 小说时间线

- **20亿年前**: 编织者文明在银河系鼎盛时期，建造星门网络基础设施。随后编织者「升维」，星渊开始作为其降维投影缓慢展开。
- **约500万年前至5万年前**: 银河系中先后有12个文明达到或超过人类当前科技水平，随后相继灭绝。它们的遗迹散布在银河系各处。最后一个灭绝文明留下了对「星渊」的警告记录，但尚未被人类发现。
- **2089年 - 第一次星际扩张**: 人类掌握核聚变推进技术，开始太阳系内的全面殖民。月球、火星、木星卫星相继建立永久定居点。
- **2415年 - 第二次星际扩张**: 曲率引擎原型机研发成功。人类首次实现超光速飞行，开始向邻近恒星系扩张。殖民时代开启。
- **2673年 - 创物质革命**: 创物质合成技术突破，第一个稳定虫洞（星门）被构建。星门网络建设开始，泛星同盟成立。
- **2834年 - 意识晶格诞生**: 铸魂者集团的前身成功实现首次完整的人类意识上传。引发了持续数十年的「何为人」伦理大辩论。铸魂者集团垄断该技术。
- **2901年 - 星渊被发现**: 银河系核心方向的深空探测器首次发现星渊异常区域。深渊学院的前身——一个学术考察团——被组建来研究这一现象。
- **2956年 - 创世碑发现**: 探险队在银河系中心附近发现了创世碑——编织者文明的实体遗迹。碑文中的部分信息被破译，揭示了星渊的超维本质。学术界震动。
- **2988年 - 边疆叛乱**: 数个边疆星区因资源分配不公和星门管控问题宣布独立，组建边缘自由联盟。叛乱被镇压，但联盟转入地下活动。韩洛·卡斯特在此期间从军。
- **3012年 - 幽荧星事故**: 深渊学院研究员苏铭在幽荧星研究中遭遇事故，失去左眼。同时获得感知星渊信号的异常能力。深渊学院在幽荧星设立秘密基地。
- **3025年 - 蜂巢计划启动**: 铸魂者集团秘密启动蜂巢计划，开始构建集体意识实验。艾瑟·莱恩的身份首次出现在集团高层记录中。
- **3044年 - 焰裔殖民地被弃**: 星渊扩张加速，泛星同盟议会投票放弃接近星渊的数个边疆殖民地。奥伦·石川的故乡赫炽星被列入放弃名单，数万焰裔族人被困。
- **3047年 - 故事开始**: 苏铭成功破译星渊信号中的第一段完整信息——编织者的实时通讯请求。沉默的觉醒正式拉开序幕。
- **建安五年·秋**: 建安五年（公元200年）秋，陈渊穿越至荆州南阳郡新野县附近。在一片荒野中醒来，被流民发现，误认为逃难的读书人。
- **建安五年·秋·第四日**: 陈渊用重结晶法在新野城南周老头处提纯精盐，名声传出。有军人（疑似刘备部下）前来，称"我家将军想见你"。
