# 化学兴汉·主线

## 主线概述

陈渊穿越至建安五年，在新野偶遇刘备。以化学知识为敲门砖，逐步赢得信任。从最基础的提纯硝石、制造黑火药开始，到改进冶炼、制造药物，一步步改变刘备集团的军事实力与经济基础。最终辅佐刘备统一天下，但他始终清醒地知道——知识改变的是武器，而真正改变命运的，是那些拿着武器的人的选择。

## 第一卷·新野蛰伏（第1-5章）

穿越降临，在新野落脚。目睹乱世惨状，决定寻找明主。以提纯食盐和制皂术为切入点，获得刘备集团注意。初次见到刘备、关羽、张飞。被诸葛亮出山前的刘备收留，成为幕僚。秘密开始火药研制。

## 第二卷·赤壁烈焰（第6-10章）

火药初次投入实战——不是作为爆炸武器，而是作为信号与纵火装置。赤壁之战中，陈渊提供了改进的火攻方案和原始燃烧弹。战后，开始系统性地建立化工作坊。

## 第三卷·入蜀之路（第11-15章）

随刘备入蜀。沿途改进冶炼技术，制造更好的兵器和盔甲。在成都建立正式的化工基地，开始研发火绳枪和手榴弹。同时用化学知识改进农业（肥料），巩固经济基础。

## 第四卷·北伐中原（第16-20章）

火器大规模投入北伐战场。战术层面的碾压带来战略优势。但陈渊越来越痛苦——他带来的每一项技术都让战争变得更高效、更致命。最终攻下洛阳，天下一统。结局是苦涩的：他改变了历史，却发现自己在这个时代永远是个异乡人。
