# 第2章

# 第二章 盐

天亮的时候，陈渊发现自己的腰快断了。

那根柱子靠了一夜，脊背上的骨头像是被人用锤子挨个敲过。他撑着地站起来，膝盖发出两声脆响，和实验室里掰断玻璃棒的声音一模一样。旁边几个流民还在睡，裹在破被褥里，像一堆被遗弃的包裹。

他走出破庙。街上已经有人了——准确地说，街上从来就有人，只是夜里缩进了门洞和墙根底下，天一亮就又长了出来。卖粥的老汉，挎篮子的妇人，蹲在路边用指甲剔牙的壮汉。这些人脸上的表情都差不多：一种灰扑扑的、提不起劲的活法。

陈渊摸了摸口袋。半截铅笔还在。衬衫口袋里还有几片碎玻璃——是那次爆炸溅进来的，他把它们小心地捏出来看了看。薄，透明，边缘锋利。

他没有扔掉它们。

玻璃在实验室里是最基本的工具。在这个时代，它同样可以是。

---

他需要三样东西：粗盐、一口锅、火。

粗盐最容易找。新野这种地方不产盐，盐是从外面运来的——要么走水路从扬州来，要么走陆路从河东解县来。不管从哪来，到了底层百姓手里，都是粗得像砂砾的灰黑色块状物，带着一股说不上来的苦涩味。那是因为里面混了大量氯化镁和硫酸钙，杂质多，味道差，但便宜。

他沿街走了大半条路，在一个巷口找到了一个卖盐的摊子。摊主是个瘦小的老头，面前铺着一块脏布，上面堆着几堆颜色深浅不一的盐块。

陈渊蹲下来，拿起一块放在鼻子下面闻了闻。

"你闻盐做甚？"老头警惕地看着他。

"看看成色。"

"成色就在那儿摆着，你要买就买，不买别动手动脚的。"

陈渊放下盐块。他口袋里没有一文钱。这一点他在昨天夜里就已经想清楚了——他在这个世界里什么都没有，但他有知识。

"老丈，"他说，"我有个法子，能把您这盐变成白的。"

老头上下打量了他一眼。眼前的年轻人穿着一件破衬衫，头发乱得像草窝，脸上还带着烟熏火燎的痕迹。怎么看都不像是个有法子的人。

"你这后生，怕不是读书读傻了？"

"我不收钱。"陈渊说，"我只要您给我一斗粗盐，一口铁锅，让我用一用。做出来的精盐，您拿去卖，我只要一成。"

老头嗤笑了一声："一斗盐做出来的白盐能有多少？你这账算得不对。"

"一斗粗盐，出三升精盐。"陈渊说，"您这粗盐卖多少钱一斤？"

"十五文。"

"精盐呢？在这新野城里，上好的白盐卖多少钱？"

老头没说话。上好的白盐，那是大户人家才用得起的。在这新野城里，一斤精盐少说也值八十文。一斗粗盐进去，三升精盐出来——他心里算了一下，眼里的警惕变成了狐疑。

"你用什么法子？"

"水煮。"

"水煮谁不会？"

"不是随便煮。"陈渊从地上捡起一根树枝，在泥地上画了一个简陋的图——一口锅，下面是火，上面画了几道波浪线表示水。"粗盐溶在水里，用布滤掉渣子，然后大火煮。煮到一定时候，水里会析出细白的盐花。把盐花捞出来，晒干。关键在于——"他用树枝点了点那个图，"水不能煮干。煮干了，杂质又回到盐里了。"

老头盯着地上的图看了半天。他一个字也不认识，但他看得懂图。种了一辈子地的人，对图形的敏感远胜过文字。

"你从哪儿学的？"

"书上看来的。"

这倒不是撒谎。重结晶法确实是写在书上的——只不过不是这个时代的书。

老头犹豫了一会儿。他这辈子见过太多说大话的人，尤其是在这种年头——今天说能点石成金的骗子，明天就能卷着你的锅和盐一起消失。但眼前这个年轻人的眼神不像骗子。骗子有热度，有迫切，有一双追着你跑的眼睛。这个人不一样。他的眼睛是凉的，像一潭水，盯着你的时候不是在看你，而是在看你背后的什么东西。

"行。"老头说，"锅我借你，盐给你一斗。做不出白盐来，你赔我一斗盐。"

"公平。"

---

老头姓周，住在城南一条窄巷子里。院子里有一口灶台，上面架着一口生了锈的铁锅。陈渊蹲在灶台前看了半天，又翻了翻灶膛里的灰——烧的柴火，不是炭。温度不够。

"有木炭吗？"他问。

"有。去年冬天剩的，不多。"

"拿来。"

周老头搬出半篓炭。陈渊挑了几块，在灶膛里架好，上面引了柴火。火舌舔上炭块的时候，他盯着那团火光出了一会儿神。

他想起实验室里的马弗炉。精确控温，数字显示屏上跳动的数字，每一摄氏度都清清楚楚。而现在他只能靠肉眼——炭火的颜色判断温度，暗红大约五六百度，亮红七八百度，发黄就上千了。

他不需要上千度。他只需要大约一百度——水的沸点。但要控制蒸发速度，不能太快。

陈渊把粗盐倒进锅里，加了水。水是从院子的井里打的，他端起来对着光照了照——不算浑，有少量悬浮物。将就用吧。

"有布吗？干净一点的。"

周老头翻出一条旧麻布，已经洗得发白了。陈渊接过来，抖了抖灰，把溶好的盐水过滤了一遍。褐色的杂质留在布上，滤过的水微微发黄，比之前清了不少。

然后他开始煮。

这是他最熟悉的动作。在实验室里，他做过上千次蒸发结晶——旋转蒸发仪、真空泵、冷凝管，一整套精密的设备配合得天衣无缝。现在他只有一口锅、一把火、一双眼睛。

但这够了。

他控制着火候，让盐水保持在微微翻滚的状态。锅里的水慢慢减少，液面开始析出一层细小的晶体。他凑近看了看——白色，半透明，颗粒均匀。

NaCl。氯化钠。化学式简洁得像一句咒语。

他用一块破碗片把结晶轻轻刮到旁边，继续煮。第一批析出的是最纯的氯化钠，杂质大多还溶在水里。只要他控制好时间，不让水煮得太干，就能得到白度很高的精盐。

周老头蹲在一旁看着。一开始他不信，后来半信半疑，再后来——他的眼睛亮了。

锅边析出的盐花是雪白的。白得不像这个世道该有的东西。

"真他娘的……"老头嘟囔了一句，声音发颤。

陈渊没搭话。他在计算。一斗粗盐大约十三斤半，按他的估计，能出三斤半到四斤精盐——前提是粗盐的纯度不低于六成。刚才过滤时看到的杂质含量，他心里已经有数了。

第一批盐出锅。他把它摊在院子里的石板上晒，白色的结晶在阳光下闪着细碎的光。周老头蹲在旁边，伸手捏了一点放进嘴里，咂了咂，眼眶居然红了。

"咸的。"他说，"光咸，不苦。"

纯粹的氯化钠就应该是这样的——只有咸味，没有杂味。苦味来自氯化镁，涩味来自硫酸钙。重结晶法把它们留在了母液里。

陈渊又煮了第二锅。

到傍晚的时候，他手里有了三斤七两精盐。白得像初雪，颗粒细而均匀，用手捻一捻就能感受到那种实验室里才有的纯度——虽然和真正的化学纯试剂相比差了十万八千里，但在这个时代，这就是奢侈品。

周老头对着那堆盐沉默了很长时间。

"你要一成。"他终于开口了。

"对。"

老头伸手，从盐堆里拨出一份，用破布包了递给他。三两七钱。

"锅你接着用。"老头说，语气和早上完全不同了。早上他是施舍，现在是巴结。"明天还来。"

陈渊接了盐，点了点头。

---

他拿着那包盐走回破庙的时候，天又黑了。街上比昨天更安静——也许是因为今天是什么忌日，也许只是因为更多的人已经吃不起晚饭了。他没有去想这些。

他在破庙里坐下，把盐包放在面前。月光照下来，白色的盐粒泛着一层淡淡的银光。

三两七钱精盐。按周老头的说法，精盐八十文一斤。三两七钱大约值十六七文钱。够买一顿粥，或者半尺粗布。

太少了。

但这是第一步。一个化学反应的第一步永远是量少——你先在试管里做，成功之后再进反应釜。他现在就在试管阶段。

他从口袋里摸出铅笔，翻过实验服，在空白处继续写：

"第二步：碱。草木灰（K₂CO₃）+ 石灰石（CaCO₃）→ 苛化 → NaOH/KOH。"
"第三步：肥皂。油脂 + 碱 → 皂化反应。"
"第四步：火药。硝石硫磺木炭，75:10:15。需硝石来源。"

他写完最后一个字，停了一下。铅笔芯在布面上留下灰色的痕迹，不如在纸上清晰，但足够辨认。

他靠回柱子上，忽然觉得饿了。今天他只喝了两碗周老头端来的稀粥——是作为"工钱"给的，粥稀得能照见碗底的裂纹。

但他没有急着睡。他在想一件事。

新野是刘备的地盘——如果历史没有偏差的话。建安五年，刘备兵败投靠刘表，被安排屯兵新野。这几年他一直在招兵买马，等待时机。一个能提纯出白雪般精盐的人，在这个时代不会被忽视太久。

问题是：他想被谁注意？

刘备？曹操？还是谁都不注意，自己苟着活？

他的脑子给了他一个答案，但他的脊背给了他另一个——昨夜靠在柱子上硌了一夜的脊背告诉他，光靠一个人的化学知识，在这个时代活不了太久。他需要一把伞，一张桌子，一个能让他的试管安稳架起来的地方。

他需要投靠一个人。

但在那之前，他得先让自己变得有用。

化学反应里有个词叫"活化能"——反应物要变成生成物，中间有一道能量壁垒。你得先输入足够的能量，反应才能启动。

他现在就在输入能量。每一天，每一粒盐，每一行写在上面的化学方程式。

---

第二天他又去了周老头那里。这一次他多要了两斗粗盐。

周老头二话没说就给了。

第三天，他出了六斤精盐。周老头开始在城里放出话去——"城南老周，新到了上等白盐，不苦不涩，入口纯咸"。

第四天，有人找上门来了。

不是买盐的。

是一个穿青布短褐的年轻男人，腰间挎着一把环首刀，站在周老头的院子门口，两个拇指钩在腰带上，看着陈渊煮盐的灶台。

"你就是那个把黑盐变白的？"

陈渊抬头看了他一眼。这人二十出头，下巴上有一道旧伤疤，目光不凶，但很稳。不是兵痞，也不是流民。这种眼神陈渊在实验室里见过——是新来报到的研究生第一次看到导师实验台时的那种眼神：好奇、警惕，还有一种压着的什么。

"是我。"陈渊说。

年轻人点了点头，转身就走。走到巷口时停了一下，扔下一句话：

"我家将军想见你。"

他没有说将军是谁。但他不需要说。

巷口站着两个同样穿青布短褐的人，腰间同样挎着环首刀。三个人形成了一个松散但有效的包围。

陈渊放下手里的碗片，擦了擦手。

锅里的盐水还在翻滚，白色的盐花正在析出，一颗一颗，安静而精确。

他点了点头："好。"
