# 第1章

# 第一章 醒来

他是被疼醒的。

确切地说，是左手手背上一片灼烧般的剧痛，把他从无边的黑暗里拽了出来。陈渊睁开眼，看见的是天。灰蒙蒙的，低矮的，像一块洗了太多遍的旧棉布，罩在头顶。

他躺在地上。泥土是湿的，带着一股腐叶和铁锈混在一起的气味。左手背上有一道三寸长的伤口，血已经干了，结成暗褐色的痂，和烧焦的衣袖粘在一起。

衣袖。

他低头看了一眼自己——身上穿的还是那件白色实验服，但已经面目全非了。前襟烧掉了一大块，露出里面同样烧焦的衬衫，左边的口袋整个不见了，右边口袋里还能摸到半支铅笔和几片碎玻璃。护目镜挂在脖子上，一片镜片裂成了蛛网状。

实验室。他想起来了。那天下午他在做硝化反应——硝酸和苯的硝化，温度控制出了问题。他记得自己伸手去关阀门，然后就是一声闷响，热浪，以及什么都看不见的白光。

然后就是现在。

陈渊慢慢坐起来。周围是一片旷野，秋天的枯草有半人高，风一吹就沙沙地响。没有建筑，没有公路，没有电线杆。远处有几棵歪脖子老槐树，树下似乎躺着什么东西，他眯着眼看了半天——是人。或者说，曾经是人。一具骸骨，已经白了大半，衣服烂成了布条，散在枯草里。

陈渊的胃猛地抽搐了一下。他在实验室里见过不少东西——浓硫酸泼在手上的味道，液氮冻伤后发黑的指尖——但那是干净的，有秩序的，写在安全手册的条款里。这不一样。这是一具真真正正的、死在荒野里无人收殓的人的骨头。

他的脑子开始自动运转了。这是它一贯的做法——遇到无法处理的事情时，就退回到最熟悉的轨道上去。

他记起一组数据：东汉末年，全国人口从五千六百万锐减至不足一千万。战争、饥荒、瘟疫，三者交织在一起，像三个互相催化的化学反应，把整个中原变成了一座巨大的坟场。

"东汉末年。"他把这四个字念出了声。

声音是真实的。风是真实的。手背上的伤是真实的。那具白骨也是真实的。

他站起来，腿有些发软。实验服在风里飘着，像一面投降的旗帜。

---

他不知道自己走了多久。太阳在头顶移了个弧度，从偏东挪到了正南。脚下的路——如果那也能叫路的话——是一条被踩出来的土道，两道车辙中间长着草。他沿着车辙走，因为想不出别的办法。

实验服他脱了，卷成一团夹在腋下。在这片旷野里穿一件白色的、烧焦的怪衣服，比不穿更引人注目。衬衫虽然也破了，但至少看起来像——他不知道像什么。像一个落魄的人吧。这一点倒是和周围的环境很协调。

他是在一条小河边遇上那群人的。

说是"一群"，其实也就十来个。男女老少都有，推着一辆独轮车，车上堆着锅碗瓢盆和几捆破被褥。女人们用布裹着头，孩子们一个比一个脏，眼睛大得惊人——不是那种天真无邪的大，而是一种吃得太少、消耗了太多之后，只剩下一对眼珠子撑着的、骨感的大。

他们看见他，停了脚步。一个黑瘦的中年男人走在最前面，手里攥着一根扁担，上下打量了他一眼。

"先生也是逃难的？"

陈渊张了张嘴，不知道该怎么回答。他确实在逃，但从什么逃开，他自己也说不清。

那男人没等他回答，就叹了口气，把扁担换了个手："这年头，谁不是逃难的。跟着走吧，前面有个县城，叫新野，听说还能讨口饭吃。"

陈渊跟着他们走了。

他没有选择。在这个世界里，他什么都没有——没有手机，没有身份证，没有钱，连一件完整的衣服都没有。他唯一拥有的，是脑子里那些化学方程式。

S + O₂ → SO₂。硫在氧气中燃烧，生成二氧化硫。

这是他学会的第一个化学方程式。初中三年级，化学课，老师写在黑板上的。全班只有他一个人在那一刻觉得，那不是方程式，而是一首诗。一种精确的、不容置疑的、美丽得近乎冷酷的诗。

现在他要用这种诗活下去了。

---

到新野的时候，天快黑了。

所谓县城，不过是一圈夯土矮墙围着的几条街。城墙上有几处豁口，用木头和杂草勉强堵着。城门口站了两个兵，穿着破旧的皮甲，手里拄着长矛，与其说是守卫，不如说是两根会喘气的木桩。

流民们排在城门外，等着被搜身放行。陈渊排在最后面，手里攥着那团实验服，心想：如果他们搜出这件衣服，自己该怎么解释？

轮到他时，那个兵瞥了他一眼。

"手。"兵说。

陈渊伸出手。兵看了看他的手——这是一双做实验的手。右手食指有一道旧伤疤，是被试管碎片划的；左手拇指和食指的指腹上有一层黄色的茧，那是长期接触硝酸银留下的痕迹。

兵显然不关心这些。他只是翻了翻陈渊的衣兜，什么也没翻着，就挥了挥手。

进城之后，流民们散了。那个黑瘦的中年男人——他好像姓魏——临走时指了指街尾的方向："那边有座破庙，能凑合一夜。"

陈渊沿着他指的方向走。街上很暗，没有路灯。两旁的房屋大多黑着灯，偶尔有一两户亮着，昏黄的火光从门缝里漏出来，像是这城市仅存的一丝体温。

他路过一家铺子，门板半开着，里面传来切菜的声音。一个老妇人蹲在门槛上，端着一碗粥，慢慢地喝。粥稀得能照见人影。

陈渊忽然觉得饿。他已经不知道自己多久没吃东西了——是爆炸之前那次午餐？还是在这条路上走了整整一天所积攒下来的、无处安放的饥饿感？

他的胃发出一声响。在安静的街上，这声音格外清晰。

老妇人抬起头，看了他一眼。那一眼里没有什么表情——不是冷漠，而是一种见过太多之后，什么都剩不下的空。她低下头，继续喝粥。

---

破庙在街尾的最尽头。说是庙，其实只是一座没有菩萨的空房子。屋顶漏了一个洞，月光从那个洞里漏下来，在地上画了一个歪歪斜斜的圆。

里面已经有人了。七八个流民缩在墙角，用干草裹着身子。看见陈渊进来，有人抬了抬眼皮，大多数人连眼皮都没抬。在这年头，一个陌生的过路人不值得多看一眼。值得多看一眼的东西——粮食、盐、一张没有被血浸透的草席——都已经太少了。

陈渊在另一边的墙角坐下。背靠着一根歪斜的柱子，把实验服铺在地上当了褥子。他的脑子很乱，但身体已经替他做了决定——先活下来，再想别的。

他从衬衫口袋里摸出那半截铅笔，在实验服的背面空白处写下了第一行字：

"硝石（KNO₃），硫磺（S），木炭（C）——配比75:10:15。"

这是黑火药的配方。他默写下来的，像在考卷上答题一样自然。

然后他停了一下，又写了一行：

"先做盐。提纯粗盐，重结晶法。这个最简单，也最安全。"

在新野这种地方，盐是硬通货。比什么都硬。他能提纯出白色的、颗粒均匀的精盐——在这个时代，这种东西和白金差不多。

他写完这两行字，把铅笔收回口袋，闭上眼睛。

月光从屋顶的破洞里落下来。落在他的脸上，落在那行化学方程式上。落在殿中那一尊已经没有头的泥塑佛像的脖颈上。

四周安静极了。远处隐约有更夫敲梆子的声音，一下，又一下，像是这个时代的心跳——缓慢的、疲惫的、不知道还能跳多久的心跳。

陈渊没有睡着。他在黑暗中睁着眼，看着头顶那片灰色的天，忽然想起了一件事。

他以前给学生上课时说过一句话："化学反应的本质，就是旧键的断裂和新键的生成。"

旧的世界在他眼前断裂了。新的还没生成。

他必须自己动手。
